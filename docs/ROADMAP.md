# Vidurai Roadmap

**Current Version:** 2.2.0-Guardian

---

## Planned for v2.3.0

### Integrations

* **Official Neovim Support:** Native Lua plugin integration for Neovim users.
* **Jupyter Notebook Support:** Cell-based context injection for data science workflows.

### Personas

* **Data Scientist Persona:** Specialized hygiene rules for large datasets and ML experiments.
* **Researcher Persona:** Academic workflow support with citation and paper context tracking.

---

## Planned for v2.5.0

* stdio-based MCP for official Claude Desktop clients
* JSON-RPC 2.0 support
* WebSocket for real-time updates
* API key authentication

---

## Planned for v3.0.0

* Multi-project workspaces
* Streaming responses
* GraphQL API
* Advanced analytics endpoints

---

*This roadmap reflects current planning and is subject to change based on community feedback.*
